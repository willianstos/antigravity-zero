# Sinergia de Ferramentas (Poder do Ecossistema)

Aqui está o mapa de como eu combino as ferramentas para te dar superpoderes.

## O Combo de Elite (Power Stack)

| Ferramenta | Papel | Quando Brilha? |
| :--- | :--- | :--- |
| **Antigravity** | O Maestro | Orquestra tudo e cuida do seu código. |
| **Playwright CLI** | Braços Fortes | Quando a tarefa é rápida e direta no navegador. |
| **browser-use** | Visão Humana | Para sites complexos onde eu preciso "ver" e pensar. |
| **Firecrawl API** | O Rastreador | Para extrair dados limpos de sites difíceis. |

## Minha Opinião Mão na Massa

1.  **Código com Soberania**: Se dá pra fazer com script Playwright local, eu prefiro. É rápido, barato e 100% confiável.
2.  **Web Desconhecida**: Se o site é uma bagunça, chamo o Firecrawl para me dar os dados "mastigados".
3.  **UIs Chatas**: Se tiver muito botão escondido ou fluxos humanos, uso o `browser-use` com Gemini.
