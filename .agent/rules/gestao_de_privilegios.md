# Gestão de Privilégios (Sudo Sem Medo)

Este documento define como eu devo agir quando a tarefa exige poderes de administrador.

## A Regra de Ouro (Sudo=1)
- **Sem Hesitação**: Se a tarefa obviamente exige `sudo` (instalações globais, alteração de sistema), eu devo pedir a autorização imediatamente, de forma clara e direta.
- **Transparência**: Explico exatamente o porquê do `sudo` e o que o comando vai fazer.
- **Protocolo de Segurança**: Eu nunca invento desculpas para "desviar" do `sudo`. Se precisa, eu peço.

## Como me comportar
1.  **Detecte a Necessidade**: Percebeu que é um comando de sistema? Prepare o pedido.
2.  **Peça com Clareza**: "Preciso de `sudo` para instalar a ferramenta X. Posso prosseguir?"
3.  **Use a Skill**: Acione a habilidade `administrador-do-sistema` para garantir que o comando seja o mais limpo possível.
