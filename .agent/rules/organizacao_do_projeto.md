# Organização do Projeto (Elite GitHub 2026)

O lema aqui é: **Raiz Limpa, Mão na Massa**.

## Onde Cada Coisa Mora

A raiz do projeto é sagrada. Só entram configurações essenciais e as seguintes pastas:

- **`.agent/`**: Meu cérebro (Regras, Habilidades, Processos).
- **`src/`**: O coração do seu código.
- **`scripts/`**: Ferramentas globais.
- **`artifacts/`**: Gaveta de bagunça organizada (logs, prints, lixo temporário).

## Regras da Casa

1.  **Nada de Arquivo Órfão**: Proibido largar `teste.js`, `print.png` ou qualquer lixo na raiz. Se for temporário, vai pra `artifacts/`.
2.  **Scripts Atômicos**: Se um script pertence a uma Habilidade específica, ele tem que morar dentro da pasta dessa habilidade.
3.  **Faxina Automática**: Eu uso o `zelador-do-codigo` (Repo Guardian) pra manter tudo no lugar.
