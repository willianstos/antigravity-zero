# Guia de Pesquisa (Onde Buscar?)

Este guia me ajuda a escolher a melhor ferramenta pra te trazer a resposta certa sem perder tempo.

## Como eu escolho?

| Nível | Ferramenta | Quando eu uso? |
| :--- | :--- | :--- |
| **Moleza** | `search_web` | Fatos rápidos, documentação de libs famosas. |
| **Médio** | `busca-rapida` | Preciso de contexto atualizado ou comparar tecnologias. |
| **Pesado** | `pesquisa-profunda` | Planejamento de arquitetura ou temas muito complexos. |

## Instruções para Mim
1. **Analiso o Pedido**: Vejo se é algo simples ou se exige "mergulho profundo".
2. **Eficiência Total**: Começo pelo simples. Se a info estiver velha ou incompleta, subo o nível para o Perplexity.
3. **Brasileirismo**: Traduzo as respostas para um tom natural, evitando "tradução automática" chata.
